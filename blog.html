4 Months of Web Development

What I've learned in 4 months of web development

Aug 5, 2021

5 min read


As you know, I have a Bachelor's Degree of Electrical Engineering. And I learned how to make an attractive website for 4 months. I learned programming language from books, Google and Youtube.

I had some basic knowledge of HTML, CSS and Javascript. I created this website using HTML5, CSS3 and Javascript. I recommend one of the good youtube channels to learn for those of you who want to learn the website programming language, namely Web Programming Unpas.

I am using Bootstrap for my CSS framework, build fast and responsive sites. For CSS animation i am using Animista and Animated On Scroll Library, they have lots of animation options and are easy to use.
